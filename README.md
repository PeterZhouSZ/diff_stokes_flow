# diff_stokes_flow
Code for the Siggraph Asia 2020 submission

## Recommended Systems
- Ubuntu 18.04
- (Mini)conda 4.7.12 or higher

## Installation
```
git clone git@github.com:mit-gfx/diff_stokes_flow.git
cd diff_stokes_flow
conda env create -f environment.yml
conda activate diff_stokes_flow
./install.sh
```
